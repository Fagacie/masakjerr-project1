# Masakjerr Recipe Management System

## Overview
A full-stack web application for managing and storing recipes.

## Problem Statement
Users need a structured platform to organize and manage cooking recipes efficiently.

## Technologies Used
- Java
- Spring Boot
- MySQL

## Features
- Add, update, delete recipes
- Database storage
- User interaction with recipes

## System Workflow
User inputs recipe → backend processes request → database stores data → frontend displays recipes.

## Results / Output
Provides a centralized system for managing recipe data.
